# Exercise 3.15 Sum

Create the method `def sum_list(numbers)` in the exercise template. The method is to return the sum of the numbers in the parameter list.

```python
numbers = []
numbers.append(3)
numbers.append(2)
numbers.append(6)
numbers.append(-1)
print(sum_list(numbers))

numbers.append(5)
numbers.append(1)
print(sum_list(numbers))
```

```plaintext
10
16
```
